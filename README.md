# Star Trek: Infinite Horizons
### An LP-style MUD built on LDMud

---

## Overview

Star Trek: Infinite Horizons is a fully playable LP-style MUD written for the LDMud driver. It features a Star Trek universe setting with 10 playable races, 5 Starfleet division classes, a WoW-style stat and equipment system, persistent character saves, and multiple hand-crafted zones.

---

## Quick Start

### Prerequisites
- Ubuntu 20.04+ / Debian 11+
- `git`, `build-essential`, `autoconf`, `bison`, `libpcre3-dev`

### Build and Run

```bash
# Clone this repo
git clone <your-repo> startrek_mud
cd startrek_mud

# Build LDMud and set up directories
./build.sh

# Start the MUD
./start.sh

# Connect via telnet
telnet localhost 4242
```

|-------------|---------------------------------|-------------------|
| Human       | +5 Presence, +3 Spirit          | Fortitude (HP regen) |
| Vulcan      | +8 Intellect, +5 Strength       | Nerve Pinch (stun) |
| Klingon     | +10 Strength, +7 Stamina, +3 Agi| Battle Fury (+25% dmg) |
| Betazoid    | +8 Intellect, +7 Spirit         | Empathic Scan (reveal stats) |
| Andorian    | +8 Agility, +6 Strength         | Antennae Awareness (+dodge) |
| Bajoran     | +8 Spirit, +5 Presence          | Prophet's Blessing (+heal recv) |
| Ferengi     | +10 Presence, +5 Agility        | Rule of Acquisition (-20% vendor) |
| Trill       | +7 Intellect, +6 Spirit         | Symbiont Knowledge (+20% XP) |
| Cardassian  | +8 Stamina, +5 Intellect        | Photographic Memory (-10% dmg) |
| Bolian      | +7 Presence, +5 Agility         | Group Motivation (buff allies) |

### Divisions (Classes)

| Division    | Primary Stats       | Armor         | Role           |
|-------------|---------------------|---------------|----------------|
| Tactical    | Strength, Agility   | Medium/Heavy  | DPS / Tank     |
| Science     | Intellect, Spirit   | Uniform       | Healer / Caster|
| Engineering | Stamina, Intellect  | Medium        | Tank / Support |
| Operations  | Agility, Presence   | Light/Medium  | Hybrid DPS     |
| Command     | Presence, Strength  | Medium        | Buffer / Leader|

### Primary Stats (WoW-style)

| Stat        | Effect                                         |
|-------------|------------------------------------------------|
| Strength    | Melee attack power (+2 AP per point), carry weight |
| Agility     | Ranged attack power, +crit, +dodge             |
| Intellect   | Max energy pool (+15 per point), tech power    |
| Stamina     | Max HP (+10 per point)                         |
| Spirit      | HP and Energy regeneration rate                |
| Presence    | Diplomacy, vendor pricing, leadership auras    |

### Secondary Stats

| Stat          | Effect                                       |
|---------------|----------------------------------------------|
| Crit Chance   | % chance to deal double damage               |
| Haste         | Reduces time between combat rounds           |
| Mastery       | Boosts class-specific mechanics              |
| Versatility   | % bonus to all damage dealt, % reduction to damage taken |
| Dodge         | % chance to fully avoid an attack            |
| Armor         | Flat damage reduction from physical attacks  |
| Energy Regen  | Energy restored per regen tick               |
| Hit Rating    | % base chance to land attacks                |

### Item Quality Tiers

| Tier      | Color    | Description             |
|-----------|----------|-------------------------|
| Poor      | Grey     | Vendor trash            |
| Common    | White    | Standard issue          |
| Uncommon  | Green    | Enhanced equipment      |
| Rare      | Blue     | Superior gear           |
| Epic      | Purple   | Exceptional items       |
| Legendary | Orange   | Storied artifacts       |

### Equipment Slots

Head, Neck, Shoulder, Back, Chest, Wrist, Hands, Waist, Legs, Feet, Finger x2, Trinket x2, Main-Hand, Off-Hand, Ranged

## Expanding the MUD

### Adding a New Room
```lpc
inherit "/std/room";
#include "/include/mudlib.h"

void create() {
    ::create();
    set_short("My New Room");
    set_long("Description here.");
    set_exit("north", "/d/myzone/rooms/next_room");
    set_zone("My Zone");
}
```

### Adding a New Monster
```lpc
inherit "/std/monster";
#include "/include/mudlib.h"

void create() {
    ::create();
    set_short("a Romulan soldier");
    add_id("romulan"); add_id("soldier");
    set_level(8);
    set_stat(STAT_STRENGTH, 20);
    set_stat(STAT_AGILITY, 18);
    recalculate_derived_stats();
    set_hp(query_max_hp());
    set_xp_value(400);
    set_faction("romulan");
    set_aggro(1);
    add_loot("/d/romulan/items/disruptor");
}
```

### Adding a New Weapon
```lpc
inherit "/std/weapon";
#include "/include/mudlib.h"

void create() {
    ::create();
    set_short("a Romulan disruptor");
    add_id("disruptor"); add_id("romulan disruptor");
    set_quality(QUALITY_UNCOMMON);
    set_weapon_type(WTYPE_DISRUPTOR);
    set_damage(12, 22);
    set_value(300);
    set_stat_bonus(STAT_AGILITY, 6);
    set_stat_bonus(STAT_CRIT_CHANCE, 2);
}
```
